## Concepts Abordés (Lien avec la L1)

Ce projet n'est pas un simple jeu ; c'est une application pratique des concepts clés qui sont, d'après mes recherches, au cœur des modules de L1 (`ip1`, `pf1`) et de S2 (POO, Structures de Données).

* **Algorithmique (`ip1`) :**
    * Implémentation d'une **boucle de jeu** principale (le "moteur" du jeu).
    * Gestion de la logique de **conditions** complexes pour les collisions (murs, corps du serpent).

* **Programmation Orientée Objet (POO) :**
    * Le projet est structuré en **Classes** et **Objets** (ex: une classe `Snake`, une classe `Food`, une classe `GamePanel`), anticipant les cours de S2.

* **Structures de Données (S2) :**
    * Utilisation d'une `ArrayList` (ou d'un tableau) pour gérer le **corps du serpent**, qui grandit et se déplace à chaque "tick" de la boucle.

* **Gestion d'événements :**
    * Capture des entrées utilisateur (clavier) pour diriger le serpent.
